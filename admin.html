Set Up a New Laptop

Follow these steps to set up a freshly installed Pop!_OS laptop.

  1. Turn on the laptop and complete the initial Pop!_OS setup (create a user account, connect to Wi-Fi)
  2. Open a terminal (press Ctrl + Alt + T or search for "Terminal")
  3. Copy and paste this command, then press Enter:
    curl -sL popos.4youth.org.uk/go | bash click to copy
  4. Wait for the setup to complete (it will install Chrome, set up shortcuts, and lock down the laptop)
  5. Restart the laptop — it's ready to go!
